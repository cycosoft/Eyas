# Technology Stack

This document outlines the primary technologies, frameworks, and libraries used in the Eyas project.

## Core Technologies
- **Programming Language:** JavaScript (ES6+) and TypeScript.
- **Runtime Environment:** Node.js.

## Application Framework
- **Desktop Shell:** Electron is used to build the cross-platform desktop application.
- **UI Framework:** The user interface for Eyas itself is built using Vue.js.
- **Build Tool:** Vite serves as the build tool and development server for the Vue.js frontend.

## Server-Side
- **Web Server:** Express.js is used to create a local server for the "Expose Test" feature.

## Testing
- **Unit & Component Testing:** Vitest is the primary framework for unit tests.
- **End-to-End (E2E) Testing:** Playwright is used for automated end-to-end testing of the application.
