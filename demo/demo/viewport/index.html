Eyas comes pre-configured with industry-standard viewports for high-fidelity testing.

Label Dimensions Aspect Ratio
Desktop 1366 × 768 16:9
Tablet 768 × 1024 3:4

Custom Breakpoints

Need to test a specific device like an iPad Pro? Just add it to your .eyas.config.js.

// .eyas.config.js
module.exports = {
  viewports: [
    { label: `iPad Pro`, width: 1024, height: 1366 }
  ]
};

Try it now

In the Browser menu, select Viewport to switch between different device sizes instantly.
