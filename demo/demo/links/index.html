Custom Menu Links

Eyas lets you add short-cut links to the app menu with built-in support for variable substitution. Great for bypassing login screens or pre-filling forms.

How variables work

You define templates in your config using {curlyBrackets}. When a user clicks the link in the menu, Eyas shows a modal to gather the required information before navigating.

Integer Substitution

Template /orders?id={int}
Result /orders?id=42

Asks the user for a number. Perfect for pre-filling specific resource IDs.

String Substitution

Template /search?q={str}
Result /search?q=test

Gathers arbitrary text from the user. Handy for search and filtering tests.

Boolean Toggle

Template /?debug={bool}
Result /?debug=true

Shows a simple true/false toggle to the user before navigating.

Select List

Template /?theme={light|dark|}
Result /?theme=dark

Generates a dropdown with specific options for the user to choose from.

Eyas Environment Variables

You can also use global environment variables prefixed with _ to dynamically resolve the current environment's domain or sub-domain.

// Example: dynamically resolving staging vs prod
{ label: `Deep Link`, url: `https://{_env.key}example.com/api` }

Try it now

Check the Test → Links menu to see these shortcuts in action within the Eyas runner.
