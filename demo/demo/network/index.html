Network Control

Eyas allows you to completely isolate your testing environment from the internet. This is useful for verifying offline fallback logic and ensuring your app stays local.

Current Network Status

📶

OFFLINE

Your browser is telling Eyas that you are disconnected from the network.

How variables work

When the network is disabled via the Developer Tools menu, Eyas intercepts and blocks all network requests (except to the internal static server).

Online Mode (Default)
All requests flow normally. Your app can fetch external resources like Google Fonts, analytics, and CDNs.
Offline Mode
All external network requests are dropped. Your local assets will continue to load, but remote URLs will fail.
