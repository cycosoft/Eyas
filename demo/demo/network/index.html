How isolation works

When "Network Isolation" is active, Eyas acts as an application-level firewall. For maximum accuracy, it severs the network stack for the entire test—including your local assets—simulating a total disconnect just like Chrome's "Offline" devtools mode.

Controlled Access
Eyas only permits requests that match your defined environment domains or local static assets. Everything else is culled.
Privacy First
Unlike a normal browser, Eyas does not share cookies or cache between different project IDs, ensuring a clean-room state every time.

Conceptual Exercise

To experience a total network disconnect, open the Development Tools menu and click Go Offline. Note that since Eyas blocks all network traffic, the test application itself will become inaccessible until you go back online.
